# 🤖 Jarvis – AI Voice Assistant (Mikasa)

## 📌 Overview

**Jarvis (Mikasa)** is an AI-powered voice assistant designed to automate tasks, interact with users through voice commands, and provide a smart desktop experience. It aims to bring a **personal assistant-like system** to your computer for productivity and convenience.

---

## 🚀 Features

* 🎤 **Voice Command Recognition** – Control system using voice
* 💻 **Task Automation** – Open apps, perform system tasks
* 🌐 **Web Integration** – Search and interact with online content
* 🔊 **Text-to-Speech (TTS)** – Assistant speaks responses
* ⚡ **Fast & Lightweight** – Efficient Python-based system
* 🧠 **AI Integration Ready** – Can be extended with AI APIs

---

## 🛠️ Tech Stack

* Python 🐍
* HTML, CSS (for UI if used)
* PowerShell Scripts
* Text-to-Speech & Speech Recognition Libraries

## ⚙️ Installation & Setup

1. Clone the repository:

```
git clone https://github.com/prashant-singh-78/jarvis.git
```

2. Navigate to the project folder:

```
cd jarvis
```

3. Create virtual environment (optional):

```
python -m venv .venv
```

4. Activate environment:

```
.venv\Scripts\activate
```

5. Install dependencies:

```
pip install -r requirements.txt
```

6. Run the assistant:

```
python app.py
```
